AppFunnel

Ruby on Rails application to manage the Code Fellows developer bootcamp application process. Prospective students apply through the website and Code Fellows staff can then review, rank and filter the applicants.

Top candidates can then be invited via email to send in a video interview and then invited into an in-person interview.

Technical skills: Ruby on Rails (Devise, email, omniauth, forms), RSpec, Git, HTML, CSS (Twitter Bootstrap), Heroku, Pair Programming

t+j Designs

I co-founded t+j designs, an ecommerce fashion jewelry website, and in less than 2 years it has reached over $350,000 in revenue and been featured on TV (The Today Show, Access Hollywood, E! News) and print (New York Post, Seventeen).

The website built on top of BigCommerce and I designed and heavily customized the theme in HTML/CSS based on existing theme. I also added some JavaScript hacks to get around limitations in BigCommerce, including the first visit sign-up modal and highlighting of sale prices and items.

Technical skills: HTML, CSS, JavaScript, BigCommerce, MailChimp
